SQL injection

SQL injection is a code injection technique where an attacker can execute malicious SQL statements in web applications database. It can affect any web application that uses a SQL database.

By leveraging SQL injection technique an attacker may use it to retrieve the contents of an entire database. This may lead to leaking sensitive data such as customer data, sensitive personal information, intellectual property and other sensitive information.

In order to exploit the vulnerability an attacker must find an input in the web application that is used directly as a part of a SQL query. The attacker may then use malicious payload as the input and execute his/her own code in the applications database.
