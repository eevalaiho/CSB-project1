Cross-Site Scripting (XSS)

Cross-site scripting (XSS) attacks are injection type attacks that are used to inject malicious scripts - generally client side scripts - in web site. An attacker may use XSS attacks to gain access to any cookies, session tokens or other sensitive information that is stored in an end-users computer or even rewrite contents of a html page.

The flawed implementation allows users to post unchecked html and javascript code in the web site's database and displays it unencoded. This allows an attacker to employ XSS attacks.
